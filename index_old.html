About Me

Projects

Advance Wars Clone

Still in early stages; proper play has yet to be implemented.
Features a complete terrain and unit set; a turn management system which allows unit selection, movement and battle (no battle animations, though); randomly generated maps; and authentically replicated minutiae from the source game. This project will eventually be unlisted for the obvious legal reasons, but the work will live on in my own project.

Movie Lookup — React/Django Test Public

A website which uses the OMDb Api to search for movies via a flashy website.
The public link is hosted by Heroku on a free account so it *will* take a moment for Heroku to spin up the server dyno on first access.
The site also has its own backend voting system. It does not have a system for user accounts or IDs, so it has no way of verifying you haven't once voted before, but alas~.

Wix Site — AddressBook

Completed for Wix certification. Keeps track of a user's personal contacts. Log in and add one, I dare you.
Users are prevented from seeing any contact which is not their own, even if they have a link to one. It's also integrated with SendGrid to allow users to send emails directly from the site to anyone in their contacts.

Budgetboy

A CLI script written in Python which, if you tell it what your bills are, will tell you when they're due forever. It has a calendar scroll and is capable of projecting long-term financial outcomes. I use it frequently to remind myself what I'm subscribed to.
